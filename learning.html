Bay-Delta SCHISM

Learning Resources for Bay-Delta SCHISM

This page is the documentation hub for running the Bay-Delta SCHISM application. It has some ideas for learning to use SCHISM and a how-to user guide for the application on the Bay-Delta domain.

For installation instructions please see the Installation and Getting Started Page.

General Resources and Tools

Tutorials Hello SCHISM tutorials are a set of SCHISM tutorials that introduce basic features, our preprocessors and can get you running the model quickly (Windows friendly).

The Bay Delta SCHISM User Guide explains many of the details of setting up and interpreting a full Bay-Delta simulation, making use of the full toolset.

Bay-Delta Tutorials and presentations describe model assumptions and model input preparation for Bay-Delta SCHISM.

Linux hints help get you started and avoid some gotchas for things like copying and archiving runs

Manuals

SCHISM Manuals are available for SCHISM at the VIMS site.

There is some documentation for our Python libraries online, including:

  • schimpy and our templates/preprocessors for grids and spatial inputs.

  • vtools3 an extension to schimpy that includes some time series manipulations we need that go beyond basic Pandas which is a prerequisite.

  • dms_datastore a library for getting data associated with the Bay-Delta.
